Prompts Guardados: Tu Recetario

¿Encontraste un prompt que funciona increíble? No lo reinventes cada vez. Guárdalo como receta y reutilízalo para siempre.

En este módulo

  • Por qué un buen prompt vale la pena guardar
  • La analogía del recetario
  • Cómo armar tus primeros

El problema

Funcionó perfecto. Y luego lo perdiste.

El mes pasado escribiste un prompt que convirtió notas desordenadas en un resumen limpio, perfecto. Hoy lo necesitas otra vez, y lo estás reescribiendo de memoria, peor que antes.

Los buenos prompts son un activo. Deja de tirarlos a la basura.

La analogía central

El Recetario

Un buen prompt es una receta: un conjunto de pasos probados que produce un buen platillo de forma confiable. No reinventarías tu mejor receta desde cero cada vez, la escribirías en una tarjeta y la guardarías en el recetario.

Los prompts guardados son tu recetario para la IA.

Se conecta justo con el Sous Chef del Nivel 1: un prompt guardado es la receta que le das a tu sous chef, lista para cocinar.

Cómo funciona en realidad

Cómo armar un recetario

Un prompt guardado (en pocas palabras: una instrucción reutilizable que conservas) es simplemente un prompt que guardas y sacas cuando lo necesitas.

Dónde viven

Una app de notas, un documento, o funciones integradas (prompts guardados, plantillas, hasta un asistente personalizado del Nivel 2).

Hazlos reutilizables

Deja espacios para llenar: “Resume [pega las notas] para [audiencia] en [largo]”. Así una receta sirve para cien platillos.

Arma un recetario chico, no una biblioteca

Cinco recetas buenas que usas cada semana le ganan a cincuenta que olvidas que tienes.

Qué significa esto para ti

Arma tu recetario

1. Guarda el próximo que funcione

Cuando un prompt te salga increíble, cópialo en algún lado antes de que se pierda para siempre.

2. Agrégale espacios para llenar

Convierte un prompt de una vez en una plantilla con [corchetes] para las partes que cambian cada vez.

3. Ten el recetario a la mano

Una nota fija o un documento de donde puedas copiar y pegar en segundos, con eso basta.

Pruébalo esta semana

La próxima vez que un prompt funcione bien, guárdalo con un par de [corchetes] para reutilizarlo en lo que sea.
