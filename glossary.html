Glossary

Every term you'll bump into, in plain words — each tied to its analogy. Tap a card to jump to the lesson.

Agent

An AI doing a task on its own — making decisions and taking steps toward a goal, not just answering a single question.

Think of it as: An employee you delegate a job to.
Learn it →
API

The messenger that lets two pieces of software talk — one app requests something from another and gets an answer back.

Think of it as: The waiter between dining room and kitchen.
Learn it →
Artifact (Canvas)

A panel beside the chat where the AI builds an editable thing — a doc, sheet, or small app.

Think of it as: A workbench next to the conversation.
Learn it →
Backend

The behind-the-scenes engine of an app — saving data, checking logins, running the logic.

Think of it as: The kitchen of a restaurant.
Learn it →
Bash

A common “language” for typing commands to a computer through a text window, instead of clicking buttons.

Think of it as: Speaking to your computer in plain commands instead of pointing and clicking.
Chat

A single conversation — quick and disposable. Nothing is kept once you move on.

Think of it as: A phone call.
Learn it →
ChatGPT

OpenAI's AI assistant — one of the most widely used, with a large free tier and many add-on features.

Think of it as: One of the “brands” of AI assistant.
Claude

Anthropic's AI assistant — the family of models this course is built around. Good at writing, analysis, and careful reasoning.

Think of it as: One of the “brands” of AI assistant.
Command

A single typed instruction you give a computer in a text window — one line that tells it to do one thing.

Think of it as: Telling a worker one specific task, out loud.
Connector

A link that lets AI reach into another app you use, like email, calendar, or Drive, to fetch or act.

Think of it as: A key to another room.
Learn it →
Context window

Everything the AI can “see” at once in a conversation. When it fills up, the oldest parts drop off.

Think of it as: A whiteboard that gets wiped when it's full.
Learn it →
Custom assistant (Custom GPT / Gem)

A helper you set up once with instructions and examples, then reuse without re-explaining.

Think of it as: A specialist you trained once.
Learn it →
Custom instructions

Standing preferences you set once that quietly apply to every chat.

Think of it as: House rules posted on the fridge.
Learn it →
Database

An organized store of an app's information, structured so any record can be found instantly.

Think of it as: A well-labeled filing cabinet.
Learn it →
Free vs. paid tier

The free plan is the same core model; paying mostly unlocks higher limits and extra features.

Think of it as: A test drive vs. leasing the car.
Learn it →
Frontend

The part of an app you see and interact with — layout, buttons, screens.

Think of it as: The dining room of a restaurant.
Learn it →
Gemini

Google's AI assistant — built into Google's products like Search, Docs, and Gmail.

Think of it as: One of the “brands” of AI assistant.
Hallucination

When AI states something false with total confidence — because it predicts likely words, not verified facts.

Think of it as: A confident intern making things up.
Learn it →
Hosting & deployment

Hosting is renting always-on space on the internet; deployment is publishing your work there so anyone can visit.

Think of it as: Renting a shop, then opening it to the public.
Learn it →
Input & output

What you send (input) and what the AI sends back (output). Both burn tokens.

Think of it as: Fuel burned on the way there and the way back.
Learn it →
Loop

A job an AI agent repeats on a schedule — it wakes up, does one defined task within set permissions, logs the result, and sleeps.

Think of it as: A worker on the night shift.
Learn it →
Memory

A few facts the AI saves and carries from one chat to the next. You can view and edit them.

Think of it as: A notebook it keeps on the side.
Learn it →
Model

The underlying AI “brain.” Newer or bigger ones are smarter but can be slower or pricier.

Think of it as: Picking the right vehicle for the trip.
Learn it →
Multimodal

AI that handles images and audio, not just text — you can show it pictures or talk to it.

Think of it as: Giving it eyes and ears.
Learn it →
Project

A saved workspace that keeps your files, standing instructions, and chats for one job together.

Think of it as: A dedicated office.
Learn it →
Prompt

What you type to the AI. The more specific it is, the better the result.

Think of it as: The recipe you hand your sous chef.
Learn it →
Rate limit

A cap on how much you can use AI within a set window of time before it makes you wait.

Think of it as: A gas station that only lets you fill up so many times an hour.
Learn it →
Saved prompt (template)

A reusable instruction you store, often with fill-in blanks, and pull out when you need it.

Think of it as: A recipe in your recipe box.
Learn it →
System prompt

Standing instructions that stay active for the whole conversation, no matter how long it runs.

Think of it as: Written in permanent marker on the whiteboard.
Learn it →
Terminal (command line)

The plain text window where you type commands to a computer directly.

Think of it as: The staff-only door to the computer — no buttons, just instructions.
Token

A chunk of a word — the unit AI actually reads and writes. Roughly three-quarters of a word.

Think of it as: Gas in the tank — every word burns a little.
Learn it →
Vibe coding

Building software by describing what you want in plain language while the AI writes the actual code.

Think of it as: Being the director, not the bricklayer.
Learn it →
